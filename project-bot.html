Descrição do Projeto

Bot de WhatsApp multifuncional chamado Ori IA, desenvolvido em Node.js com suporte à API do WhatsApp. Ele oferece várias funcionalidades para facilitar o dia a dia dos usuários, como responder perguntas, gerar textos, criar imagens, ajudar nos estudos, gerar QR codes, imitar vozes e muito mais.

Principais características

  • Respostas automáticas com IA
  • Criação de currículos e capas
  • Gerador de QR Code e áudio
  • Leitor e imitador de voz
  • Ajuda escolar e explicações personalizadas
  • Geração de imagens e análise de fotos
  • Interação via comandos simples pelo WhatsApp

Tecnologias utilizadas

whatsapp-web.js
Node.js
ChatGPT API
